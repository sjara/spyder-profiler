 ___________________

   spyder-profiler
 ___________________


This project provides a profiler plugin for the python IDE Spyder.


INSTALLATION
------------

1. Place  p_profiler.*  in
   SPYDERPATH/spyderlib/spyderplugins/
2. Place  profilergui.py  in
   SPYDERPATH/spyderlib/spyderplugins/widgets/


DOCUMENTATION
-------------

Screenshots and documentation can be found at:
http://sjara.github.com/spyder-profiler/
